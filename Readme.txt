Please load Function.R before doing the simulation and real data analysis. Each folder has the code listed below: 

1. R package: MVOPR r package in CRAN
2. Real Data Analysis: Code and data for CAARS Real Data Analysis
3. Simulation.1: Simulation code for MVOPR for two modalities 
4. Simulation.2: Simulation code for MVOPR for misspecified  case
5. Simulation.3: Simulation code for MVOPR for three modalities
6. Function.R: Code to use MVOPR, Factor based model, RRR, and data generation
